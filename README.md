# Betel's 25th Birthday Gift

This is a small, interactive website created as a 25th birthday gift for Betel from Kiara. 
It serves as a digital invitation for a joint nail set appointment (a "free.99" set chosen by Betel, valid between 04/04 - 04/07), presented with a cute 90s/2000s nostalgic aesthetic.

## Features

- **Interactive Postcard:** A breathing envelope that opens to reveal a 3D flippable digital coupon.
- **2000s Aesthetic:** Pink, cream, and white color palette with nostalgic elements like a spinning music widget and a diary lock authentication page.
- **Personal Archive:** A magazine-style layout for journaling and memories, complete with column-based entries and a typing effect.
- **Mobile Optimized:** Designed specifically to look great and feel responsive when sent via text and opened on a smartphone.

## Project Structure

- `index.html`: The main postcard invitation page.
- `auth.html`: The diary lock/password protection page.
- `home.html`: The personal archive/magazine layout page.
- `assets/`: Contains the CSS, JavaScript, and image files.
  - `css/`: Contains the stylesheets.
    - `index.css`, `home.css`, `auth.css`, `style.css`
  - `js/`: Contains the JavaScript files.
    - `main.js`, `posts.js`, `auth.js`
  - `images/`: Contains the nostalgic images, background, and envelope assets.
